Tentang App Links

Halaman ini berfungsi sebagai:

  • Hosting untuk file assetlinks.json yang diperlukan untuk verifikasi App Links
  • Halaman fallback jika pengguna tidak memiliki aplikasi terinstal
  • Halaman informasi tentang aplikasi Anda

Catatan untuk Developer:

File assetlinks.json harus diakses via HTTPS di:

https://domain-anda.com/.well-known/assetlinks.json

Pastikan file tersebut mengikuti format yang benar untuk verifikasi App Links.
